# FreeOS
FreeOS is an open-source operating system for $3 CPU chips

https://chrisg.co/unreleased-source/freeos/index.php

To develop FreeOS today from scratch, tech giants like Apple, Microsoft, or Google would spend hundreds of millions of dollars in R&D. Developing nations, lacking immense capital, might never afford to engineer their own native mobile OS. Fortunately, FreeOS was made by a developer who wrote hundreds of thousands of lines of Objective-C code by hand for iOS and macOS before architecting this custom system with the help of AI.

Now, that price is lowered to free.

Simply download FreeOS software and PCB designs, order LCD components, send PCB design to any of many PCB manufacturing factories, and you will have a $10 pocket computer, when no pocket computer devices are sold anywhere in the world for under $20 at the lowest.

Palm Pilot, Windows Mobile, Nokia Symbian... Those three pocket computer makers shut down and abandoned those platforms almost two decades ago. If they never stopped maintaining the software sources of those platforms, they would be able to sell $10 pocket computers or smartphones in the current year running the elite, highly luxurious software of two decades ago that was only for the richest people of 2006.

Nobody, anywhere, until FreeOS made a pocket computer for $10, even though it is possible.
