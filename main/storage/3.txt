Do you know any esp32 code to print how much available program RAM is left on the system

Yes, you can determine the free memory available on an ESP32 using the `heap_caps_get_free_size()` function provided by the ESP-IDF. This function can tell you how much free heap space is available, and by specifying different capabilities, you can query different types of memory, such as general-purpose, DMA-capable, etc.

Here's a simple example to print out the amount of free heap memory available on the ESP32:

```c
#include <stdio.h>
#include "freertos/FreeRTOS.h"
#include "freertos/task.h"
#include "esp_system.h"
#include "esp_heap_caps.h"

void app_main()
{
    // Print free heap memory information
    size_t free_heap_size = heap_caps_get_free_size(MALLOC_CAP_8BIT);
    printf("Free heap size: %zu bytes\n", free_heap_size);

    // Optionally, you can print other heap info
    size_t free_internal_heap_size = heap_caps_get_free_size(MALLOC_CAP_INTERNAL);
    printf("Free internal heap size: %zu bytes\n", free_internal_heap_size);

    // Loop to periodically print the information
    while (true) {
        printf("Free heap size: %zu bytes\n", heap_caps_get_free_size(MALLOC_CAP_8BIT));
        vTaskDelay(pdMS_TO_TICKS(1000));  // Wait for 1 second
    }
}
```

   - This function is used to get the free size of the heap memory with specific capabilities.

3. **Continuous Monitoring**:
   - A loop with `vTaskDelay` periodically checks and prints the free heap size every second, which is useful to observe changes over time as the program executes.

By running this example on your ESP32, you can see how much available RAM you have, which is critical for tasks such as debugging memory usage in embedded applications.
